# Neon Red Terminal for Kali Linux 🩸

A simple bash script that transforms your Kali Linux GNOME Terminal with an eye-catching neon red text color, perfect for a cyberpunk/hacker aesthetic.

## Features
- 🔴 Sets terminal text to vibrant neon red (#FF073A)
- ⚫ Configures dark background for optimal contrast
- 🖱️ Matches cursor color to neon red
- 🔄 Includes reset script to restore defaults
- ⚡ One-liner version also available

## Quick Start
```bash
git clone https://github.com/hash-089/neon-red-terminal
cd neon-red-terminal
chmod +x set_neon_terminal.sh
./set_neon_terminal.sh
Requirements
Kali Linux (or any distro with GNOME Terminal)

dconf-cli (auto-installed by script)
